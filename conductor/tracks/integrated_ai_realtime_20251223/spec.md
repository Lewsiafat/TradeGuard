# Track Spec: 整合實時報價顯示與基本 AI 行情分析框架

## 1. 概述
本追蹤旨在建立 TradeGuard 的實時數據基礎設施，並串接初步的 AI 行情分析功能，讓用戶能即時掌握市場動向並獲得數據驅動的投資建議。

## 2. 功能需求
- **實時報價顯示**：
    - 整合加密貨幣行情 API (如 Binance WebSocket)。
    - 在 UI 固定位置顯示主要交易對的實時價格。
    - 價格更新時需具備微動效 (紅/綠閃爍)。
- **AI 分析框架**：
    - 建立 AI 分析服務模組，支援串接 Gemini 或類似 API。
    - 實作「按需分析」模式，用戶點擊後生成市場報告。
    - 分析內容需包含技術指標摘要與風險評估。

## 3. 技術設計
- **前端實作**：使用 React Hooks 處理 WebSocket 連線與數據更新。
- **狀態管理**：行情數據應具備高效的更新機制，避免不必要的重新渲染。
- **測試策略**：針對數據處理邏輯編寫單元測試，模擬 API 回應。

## 4. 驗證標準
- 用戶能在介面看到跳動的實時價格。
- 用戶點擊分析按鈕後，能在 10 秒內看到 AI 生成的結構化報告。
- 程式碼測試覆蓋率需 >80%。
